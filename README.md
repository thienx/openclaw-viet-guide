## 🤖 OpenClaw là gì?

[OpenClaw](https://openclaw.ai) là phần mềm cho phép bạn tạo AI bot cá nhân sử dụng Claude (Anthropic) làm brain. Bot có thể:

- 💬 Chat 24/7 qua Telegram
- 📸 Phân tích ảnh (Vision AI)
- ⏰ Nhắc nhở tự động (Cron jobs)
- 📊 Ghi dữ liệu Google Sheets
- 🌐 Tìm kiếm web realtime
- 🔧 Tự quản lý file & hệ thống

## 🚀 Chạy guide này locally

```bash
# Clone repo
git clone https://github.com/USERNAME/openclaw-viet-guide.git
cd openclaw-viet-guide

# Cài dependencies
npm install

# Chạy development
npm start

# Build production
npm run build
```

---

## 📦 Deploy lên GitHub Pages

```bash
# Cài gh-pages
npm install -g gh-pages

# Sửa homepage trong package.json
# "homepage": "https://USERNAME.github.io/openclaw-viet-guide"

# Deploy
npm run deploy
```
